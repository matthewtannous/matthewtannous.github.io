Real-time gaming web application using React and NestJS, integrating RESTful APIs with web sockets to maintain synchronized game state across clients.
Users can log in, challenge each other to classic games such as tic-tac-toe and connect 4, and play in real-time using web sockets.
The game state is saved in a database after every move, so players can choose to make a move at any time.
